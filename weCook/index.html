weCook - Graduate Project

weCook is a social recipe website, designed to make it easy to follow and edit recipes. Recipes added to the site are open to editing by all users, allowing for multiple versions of the recipe to be made.

11 March 2017

weCook was a cakePhp project I worked on for my Graduate Project. I created a social recipe website which allowed users to add recipes and edit other users recipes in an attempt to create new and exciting alternatives to users recipes and promote creativity through social interaction.
